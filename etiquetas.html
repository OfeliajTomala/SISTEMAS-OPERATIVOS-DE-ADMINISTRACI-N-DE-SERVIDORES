USO DE ETIQUETAS HTML

ETIQUETAS BÁSICAS

<html>

Es la etiqueta raíz que envuelve todo el contenido HTML de la página web. Señala al navegador que lo que contiene es código HTML.
Ejemplo:

<html>
<-- contenido HTML aquí -->
</html>

<head>

Contiene metadatos e información sobre el documento, como título, enlaces a hojas de estilo, scripts, y configuración para el navegador, pero no se muestra directamente en la página.

<head>
<title>Mi Página Web </title>
</head>


<body>

Incluye todo el contenido visible de la página que se muestra al usuario, como texto, imágenes, videos, etc.
Ejemplo:

<body>
<h1>Bienvenidos a mi web</h1>
<p>Este es el contenido</p>
<body>

Bienvenido a mi web

Este es el contenido


<h1> a <h6>

Representan títulos y subtítulos, donde <h1> es el de mayor nivel (más importante) y <h6> el de menor.
Ejemplo:

<h1>Título principal</h1>
<h2>Subtítulo</h2>

Título principal


Subtítulo


<p>

Define un párrafo de texto.
Ejemplo:

<p>Este es un párrafo de texto en la página web.</p>

Este es un párrafo de texto en la página web.


<a>

Etiqueta para crear enlaces o hipervínculos a otras páginas o recursos.
Ejemplo:

<a href="https://ofeliajtomala.github.io/SISTEMAS-OPERATIVOS-DE-ADMINISTRACI-N-DE-SERVIDORES/">Página principal</a>
Página principal

<img>

Sirve para insertar imágenes en la página. Usa atributos como src para la ruta y alt para texto alternativo.
Ejemplo:

<img src="imagenesetiquetas/imagen.jpeg" alt="Foto de ejemplo" width="300">


<ul>

Define una lista desordenada, es decir, una lista con viñetas.
Ejemplo:

<ul>
<li>Elemento 1</li>
<li>Elemento 2</li>
</ul>

  • Elemento 1
  • Elemento 2

<ol>

Define una lista ordenada, numerada.
Ejemplo:

<ol>
<li>Primer paso</li>
<li>Segundo paso</li>
<ol>
  1. Primer paso
  2. Segundo paso

<table>

Sirve para crear y definir una tabla en la página web. Esta tabla permite organizar datos en filas y columnas, mostrando información de forma clara y estructurada para facilitar su lectura y comparación.
Ejemplo:

<table>
<tr>
<th>Nombre</th>
<th>Edad</th>
</tr>
<tr>
<td>Ana</td>
<td>28</td>
</tr>
<tr>
<td>Carlos</td>
<td>35</td>
</tr>
</table>

Nombre Edad
Ana 28
Carlos 35

ETIQUETAS NUEVAS

<video>

Sirve para insertar y reproducir videos directamente en la página web. Incluye controles para reproducir, pausar y ajustar el volumen.
Ejemplo:

<video width="320" height="240" controls >
<source src="imagenesetiquetas/ejemplo.mp4" type="video/mp4">
</video>

<audio>

Permite incluir archivos de audio con controles para reproducir, pausar y ajustar el volumen.
Ejemplo:

<audio controls>
<source src="audio.mp3" type="audio/mpeg">
</audio>


<canvas>

Es un contenedor donde se pueden dibujar gráficos y animaciones dinámicamente usando JavaScript.
Ejemplo con canvas (JS externo):

<canvas id="miCanvas" width="370" height="200" style="border:1px solid black;">
<!-- Vinculamos el archivo JS -->
<script src="script.js"></script>


<figure>

Se usa para agrupar contenido multimedia (como imágenes o videos) y su leyenda relacionada.

<figcaption>

Etiqueta que se utiliza para añadir un título o descripción a un elemento <figure>
Ejemplo:

<figure>
<mg src="imagenesetiquetas/imagen.jpeg" alt="Descripción de la imagen" width="300">
<figcaption>Esta es una imagen de ejemplo</figcaption>
</figure>


<progress>

Muestra una barra visual que indica el progreso de una tarea, por ejemplo, descarga o carga.
Ejemplo con <progress> (JS en archivo aparte)

<label id="porcentaje">0% completado</label>
<progress id="miProgress" value="0" max="100">0%</progress>
<button onclick="iniciarCarga()">Iniciar carga</button>
<!-- Vinculamos el archivo JS externo -->
< script src="script.js"></script>

0%

<meter>

Representa un valor numérico dentro de un rango conocido, útil para mostrar niveles, porcentajes o medidas.
Ejemplo:

<label for="nivel">Nivel de batería:</label>
<meter id="nivel" value="0.6" min="0" max="1" low="0.2" high="0.8" optimum="0.7"> 60% </meter>
60%

<details>

Crea un widget desplegable que el usuario puede abrir o cerrar para mostrar u ocultar información adicional.

<summary>

Define el título o encabezado visible para la etiqueta <details>. Se hace clic en él para desplegar el contenido.

<details>
<summary>Más información <summary>
Aquí se muestra la información adicional oculta.
<details>

Más información Aquí se muestra la información adicional oculta.

<mark>

Resalta texto con un fondo amarillo (por defecto), útil para marcar resultados o información importante.

<p>Este texto contiene una<mark>palabra resaltada</mark>para llamar la atención.</p>

Este texto contiene una palabra resaltada para llamar la atención.
